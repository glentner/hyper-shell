Arguments
^^^^^^^^^

ID
    Unique task UUID.

Options
^^^^^^^

``--json``
    Format metadata output as JSON.

``-x``/``--extract`` *FIELD*
    Print this field only (e.g., ``-x submit_time``).

``--stdout``
    Print <stdout> of task if captured, fetch from client if necessary.

``--stderr``
    Print <stderr> of task if captured, fetch from client if necessary.
